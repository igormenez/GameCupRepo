<h2 id="instalacao-api">Instalação/Execucão</h2>
 
          cd Backend/GameCup

          dotnet build
          
          dotnet run

<h2 id="teste-api">Testes</h2>
 
          cd Backend/GameCup.Test

          dotnet build

          dotnet test

<h2 id="instalacao-front">Instalação/Execucão</h2>

          cd Frontend/GameCupFront
 
          yarn install

          yarn start
